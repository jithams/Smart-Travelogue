# Smart Travelogue

Smart Travelogue is a travel assistance and travel experience sharing application developed as an MCA final year project. The system helps travelers discover tourist attractions, nearby hotels, hospitals, and travel packages while allowing users to share their travel experiences through travelogues, videos, and descriptions.

## Project Overview

Traveling to unfamiliar places can be challenging. Smart Travelogue provides location-based travel information and a platform where travelers can share experiences and recommendations. The application aims to make travel safer, easier, and more informative.

## Features

### Admin Module
- Manage tourist places and place categories
- Add and manage hotels and travel packages
- Manage notifications
- View user feedback
- Monitor travelogue submissions
- Publish user travelogues

### User Module
- User registration and login
- View tourist attractions
- View nearby hotels and packages
- View nearby hospitals
- Upload travelogues with images
- Share travel experiences
- View notifications
- Submit feedback

### Public Module
- View tourist places
- Explore travel experiences
- Access shared travelogues
- View public travel content

## System Architecture

The system consists of three major modules:

1. Admin
2. Registered Users
3. Public Users

The backend API is developed using Flask, while the mobile application is built using Flutter. Data is stored and managed using MySQL.
